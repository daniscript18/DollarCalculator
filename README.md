# Dollar Calculator 
 - Español

Con esta calculadora podrás realizar operaciones básicas de acuerdo al dolar venezolano, este pequeño script extrae el precio del dólar de la página de DolarToday y luego ejecutando el código podrás seleccionar entre dos opciones que harán cálculos rápidos.

- English

With this calculator you will be able to carry out basic operations according to the Venezuelan dollar, this small script extracts the price of the dollar from the DolarToday page and then by executing the code you will be able to select between two options that will make quick calculations.

## Ejecución / Performance

```bash
python3 main.py
```
